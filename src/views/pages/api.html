API Documentation

Explore how to integrate with our API.

For questions or support, contact us via email at mail@jaw.dev or report an issue on GitHub.

Method Endpoint Description
GET /api/categories Get all categories with their carriers
GET /api/categories/:name Search categories by name
GET /api/carriers Get all carriers with their email domains
GET /api/carriers/:id Get specific carrier by ID
GET /api/emails Get all carrier email domains

Categories

GET /api/categories

Retrieve all categories with their associated carriers.

Response Format:
{
  "data": [
    {
      "id": 1,
      "name": "Category Name",
      "carriers": [
        {
          "id": 1,
          "name": "Carrier Name"
        }
      ]
    }
  ]
}

GET /api/categories/:name

Search for a specific category by name (supports partial matches).

Parameters:

name - Category name to search for

Response Format:
{
  "data": {
    "id": 1,
    "name": "Category Name",
    "carriers": [
      {
        "id": 1,
        "name": "Carrier Name"
      }
    ]
  }
}

Carriers

GET /api/carriers

Retrieve all carriers with their associated email domains.

Response Format:
{
  "data": [
    {
      "id": 1,
      "name": "Carrier Name",
      "emails": [
        {
          "id": 1,
          "email": "sms@carrier.com"
        }
      ]
    }
  ]
}

GET /api/carriers/:id

Retrieve a specific carrier by ID with its email domains.

Parameters:

id - Carrier ID

Response Format:
{
  "data": {
    "id": 1,
    "name": "Carrier Name",
    "emails": [
      {
        "id": 1,
        "email": "sms@carrier.com"
      }
    ]
  }
}

Email

GET /api/emails

Retrieve all carrier email domains.

Response Format:
{
  "data": [
    {
      "id": 1,
      "email": "sms@carrier.com"
    }
  ]
}
